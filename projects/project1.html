포트폴리오로 돌아가기

삼성라이온즈 팬 관리 서비스

2025.05 - 2025.07 백엔드 개발

프로젝트 개요

전자정부 프레임워크 기반의 레거시 시스템을 Node.js 환경으로 전환하는 작업을 수행했습니다. 기존 Java 기반 시스템을 현대적인 TypeScript + NestJS 환경으로 마이그레이션하여 개발 효율성과 시스템 성능을 크게 향상시켰습니다.

사용 기술

TypeScript Node.js NestJS TypeORM Java Spring Boot Docker Jenkins MSSQL

참여 인원

백엔드 1명
프론트엔드 2명
앱 개발 2명

주요 업무

🔧

전체 기능 개발

회원가입, 로그인, 이벤트 참여/추첨, 쿠폰 발급, 약관 동의, 앱 푸시 알림 등 전체 기능 개발

🗄️

데이터베이스 재설계

기존 데이터베이스 구조 분석 후 TypeORM 기반으로 재설계 및 인덱스 최적화

🚀

CI/CD 파이프라인 구축

Docker + Jenkins 기반 개발 서버 무중단 자동 배포 파이프라인 구축

🏗️

인프라 설계

폐쇄망 운영 서버 환경 구축 및 인프라 설계, 외부 서비스 연동을 위한 실무 담당자와의 협업

📚

배포 가이드 작성

폐쇄망 수동 배포 시 실수 방지를 위해 배포 가이드 문서 작성 및 팀 공유

주요 성과

✅

레거시 시스템 성공적 전환

전자정부 프레임워크 기반의 레거시 시스템을 Next.js, NestJS 기반으로 성공적 전환

⚡

배포 시간 70% 단축

Tomcat WAR 배포 방식 대비 PM2 기반 클러스터 구조로 전환하여 배포 시간 10분 → 3분으로 최대 70% 단축

📊

데이터베이스 구조 간소화

도메인 중심 설계를 적용하여 기존 235개의 테이블을 38개로 구조 간소화 및 관리 복잡도 감소

🛡️

중복 참여 오류 100% 차단

이벤트 참여 시 고유 참여 키 생성 및 유니크 제약 조건을 설정하여 중복 참여 오류 100% 차단

🎯

비즈니스 로직 단순화

불필요한 예외 처리를 정리하여 비즈니스 로직의 흐름을 단순화하고 사용자 불편 최소화

🚀

대용량 데이터 처리 개선

대용량 데이터 처리 시 일괄 조회 방식에서 제네레이터 기반 chunk 단위 조회 방식으로 개선하여 서버 안정성 및 신뢰성 향상
