프로젝트 개요

전자정부 프레임워크 기반의 레거시 시스템을 Node.js 환경으로 전환하는 작업을 수행했습니다.
기존 Java 기반 시스템을 Node.js 환경으로 마이그레이션하여 개발 효율성과 시스템 성능을 크게 향상시켰습니다.

사용 기술

TypeScript Node.js NestJS TypeORM Java Spring Boot Docker Jenkins MSSQL

개선 사례

배포 구조 개선

📍 기존 상황

전자정부 프레임워크 기반 WAR 파일 9개를 순차적으로 배포하고 있었고, 배포 시 Nginx 설정에서 배포 중인 서비스를 주석 처리 후 WAR를 배포해야 했었음.

⚠️ 문제점

  • WAR 배포 시 Nginx 설정을 반복해서 수정해야 하고 실수로 인해 서비스가 중단되는 상황이 발생하였음
  • 배포 시 서비스를 내리고 WAR를 교체한 후 다시 서비스를 올리는데 대략 2분 정도가 소요되었고 9개 배포 시 시간이 너무 많이 소요되는 문제가 있었음

🛠 해결 방안

  • Node.js 환경으로 전환하면서 PM2를 사용하여 서비스를 내렸다 올리는 번거로움 없이 클러스터 모드로 무중단 배포 환경을 구성하였음
  • 명령어로 간단하게 배포할 수 있게 되어 실수가 발생할 확률이 줄어들었고 배포 시간 또한 기존 15분이 넘는 시간에서 3분으로 단축하였음

🎯 개선 결과

  • 배포 시간 단축 (15분 → 3분)
  • 무중단 배포로 서비스 가용성 향상
  • 개발 및 유지보수 효율성 크게 향상 (배포 시간 단축으로 개발 효율성 향상)
